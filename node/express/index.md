## express安装

1. 安装脚手架
```
npm install express-generator -g
```
2. 初始化项目
```
express projectName
```
3. npm install
4. npm run start 
5. start 命令优化
- 安装依赖
```
npm install nodemon cross-env --save-dev
```
- 修改启动命令
```
    "cross-env NODE_ENV=dev nodemon ./bin/www"           // 开发环境启动
```
